# PowerBI-Sales-Performance-Dashboard
# Sales Performance Dashboard | Power BI

## Project Overview

This project is an interactive Sales Performance Dashboard developed using Power BI. The dashboard provides insights into sales, profit, regional performance, and product-level analysis through multiple report pages and drill-through functionality.

## Objectives

* Analyze overall sales and profit performance
* Identify top-performing and low-performing products
* Evaluate regional sales trends
* Enable detailed regional analysis through drill-through reports

## Tools Used

* Power BI
* Power Query
* DAX
* Excel

## Dashboard Pages

### 1. Executive Overview

* KPI Cards
* Monthly Sales Trend
* Sales by Region
* Sales by Category

### 2. Profitability Analysis

* Profit by Category
* Profit by Sub-Category
* Top Products
* Bottom Products

### 3. Regional Performance

* Regional Sales Analysis
* Regional Profit Analysis
* Regional Comparisons

### 4. Regional Details (Drill-through)

* Region-specific detailed transaction view
* Interactive drill-through navigation

## Key Features

* Interactive slicers
* Drill-through functionality
* Dynamic filtering
* KPI reporting
* Business performance monitoring

## Skills Demonstrated

* Data Modeling
* Data Visualization
* DAX Measures
* Power Query Transformations
* Dashboard Design
* Business Intelligence Reporting
